# IN WC-CRM Extensions
## Дополнения и расширения плагина WooCommerce Customer Relationship Manager
Версия 0.1

[WooCommerce Customer Relationship Manager](https://codecanyon.net/item/woocommerce-customer-relationship-manager/5712695)
CRM с нужными функциями. Данный плагшин добавляет новые фунции, интерфейс и шорткоды.

Реализует шорткоды:
* incrm_clients 	- Вывод таблицы клиентов
	* title		- Заголовок перед таблицей клиентов
